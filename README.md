### Installation & Usage

1. Clone the repo
   ```sh
   git clone https://github.com/Swafox/Minter-Treasurer
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```js
   npm server.js
   ```
